# Extra Commands
## Commands
```
2..
/ability (get|modify) <single player selector: target> ...
Query or modify player abilities.

2..
/data (get|merge|modify|remove) item <slot: slot> ...
Data commands for items.

2..
/effect clear
Clear your own effects.

2..
/execute ... if data item <coordinates|single selector: target> <slot: slot> <nbt path|nbt: path> ...
Condition for item data.

2..
/execute ... if item <coordinates|single selector: target> <slot: slot> <item: item> <range: count> ...
Condition for items.

2..
/execute ... sudo ...
DETROIT BECOME COMMAND BLOCK
```
